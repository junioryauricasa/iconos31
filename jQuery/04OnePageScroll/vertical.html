jQuery es una librería JavaScript, creada inicialmente por John Resig, que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web. Fue presentada el 14 de enero de 2006 en el BarCamp NYC. jQuery es una de las librerías JavaScript más utilizadas.

jQuery es una librería JavaScript rápida, pequeña y rica en funciones. Más simple de usar que cualquier API, sirve para recorrer y manipular el DOM de HTML, manejo de eventos, animación, AJAX y mucho más en cualquier navegador web. Con una combinación de versatilidad y capacidad de ampliación, jQuery ha cambiado la forma en que millones de personas escriben JavaScript.

jQuery UI es un conjunto de componentes interactivos con la interfaz de usuario, efectos, widgets y temas incorporados en la parte superior de la librería jQuery. Si estás construyendo aplicaciones web altamente interactivas o sólo tienes que añadir un selector de fecha para un control de formulario, jQuery UI es la elección perfecta.

jQueryMobile es un framework optimizado para dispositivos táctiles, es un sistema de interfaz de usuario basado en HTML5 diseñado para hacer sitios web sensibles y aplicaciones accesibles en todos los dispositivos: teléfonos inteligentes, tabletas y computadoras de escritorio.

QUnit es un conjunto de pruebas unitarias fácil de usar que en la actualidad es usado por los proyectos JQuery, JQuery UI, JQuery Mobile y además es capaz de hacer pruebas de cualquier código Javascript, incluyéndose a sí mismo.

Sizzle es, básicamente, un motor de selectores hecho en Javascript, bastante útil para proyectos pequeños que no requieran todas las funcionlidades de jQuery.

Sizzle es muy fácil de usar:

Sizzle('.header');

Así selecconamos nuestro elemento con clase header. Similar a jQuery, podemos hacer esto también:

var s = Sizzle;
s('.header');
